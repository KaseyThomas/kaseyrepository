Instructions
  1. Open Illustrator
  2. open a new 8.25 x 8.25 file that has 1 artboard
  3. Use the polygon tool to make a polygon
  4. Use the selection tool to select the polygon you've made
  5. In the properties tab, in the appearance section change the stroke value to 1 pt
  6. In the properties tab, in the appearance section make sure the "fill" is white
  7. In the properties tab, in the transform section change the X value to 4.26 inches
  8. In the properties tab, in the transform section change the Y value to 3.877
  9. In the properties tab, in the transform section change the W value (width) to 1.69 inches
  10. In the properties tab, in the transform section change the H value (height) to 1.584
  11. In the properties tab, in the transform section change the angle (where the angle symbol is. It is right underneath the Y value) to 253.49 degrees
  12. Make sure all values are correct
  13. Now choose file > Export > Export to screens
  14. Set "Export to" as your desktop
  15. Set the file format to a PNG
  16. Set the scale to 1x
  17. Then click "Export Artboard"
  18. Email your PNG to: kthoma37@masonlive.gmu.edu
